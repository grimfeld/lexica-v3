# No engagement mechanics

The app deliberately omits the standard engagement-maximizing patterns:
no streaks, no daily goals, no "days active" framing, no punitive overdue
counts, no escalating notifications.

- Reminders: a single, opt-in, user-set daily reminder that fires
  unconditionally as a plain habit cue ("time to practice?"), never a
  due-count nag.
- Backlog: present but never weaponized — no guilt-inducing overdue number.
  FSRS tolerates late reviews.
- Stats: retention-oriented only (retention rate, totals, per-Deck progress,
  neutral review history). No engagement-oriented stats (streaks, time-in-app,
  goal completion).

This realizes the north star: the app is a tool the user picks up, not a
driver that summons them. The absence of these features is intentional — a
future contributor should not "fix" their omission by adding them back.
